# 🎮 GameLog

**GameLog** é uma plataforma social de rastreamento, avaliação e descoberta de videogames, inspirada no [Letterboxd](https://letterboxd.com).

> *Track every game. Share every thought.*

## Stack

- **Framework:** Next.js 14 (App Router)
- **Estilização:** Tailwind CSS
- **API de jogos:** IGDB (via Twitch API)
- **Deploy:** Vercel

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
